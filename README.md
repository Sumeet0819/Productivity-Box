# Productivity Box

Productivity Box is a lightweight personal productivity web app designed to help users manage their daily focus and tasks.

## Overview

This project brings together a set of useful productivity tools in one place:
- **Calendar** for tracking important dates and planning your schedule
- **Pomodoro timer** for focused work intervals and short breaks
- **Todo list** for managing tasks and staying organized
- **Weather widget** for quick local weather updates and planning

## Use Cases

Productivity Box is ideal for:
- Students organizing study sessions and assignments
- Professionals managing daily tasks, meetings, and focused work blocks
- Anyone who wants a simple dashboard to keep their day on track

## Project Structure

- `frontend/` - contains the web application source code
  - `src/components/calender` - calendar component
  - `src/components/pomodoro` - Pomodoro timer component
  - `src/components/todo` - Todo list component
  - `src/components/weather` - Weather display component
  - `src/services/weatherService.js` - weather data utility

## Getting Started

To run the app locally:

```bash
cd frontend
npm install
npm run dev
```

Then open the local development URL shown in the terminal.

## Intended Workflow

1. Open Productivity Box in your browser.
2. Add and manage tasks in the Todo component.
3. Use the Pomodoro timer to work in focused intervals.
4. Reference the Calendar for scheduling.
5. Check the Weather widget before planning outdoor activities.

## Notes
